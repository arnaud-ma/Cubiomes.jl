Getting Started ​

If you are already familiar with Julia, you can skip this section and go directly to the guide.

To install Julia and learn how to run Julia code, refer to the Getting Started section of the Julia documentation.

Installation ​

To install Cubiomes.jl, start up Julia and enter the following command in the REPL:

julia
julia> using Pkg
julia> Pkg.add(url="https://github.com/arnaud-ma/Cubiomes.jl")

Then, you can import Cubiomes.jl into the namespace

julia
julia> using Cubiomes

Getting help ​

julia
